# 📚 Lista de Contatos APP

>Status do projeto : Em desenvolvimento

## 📋 Descrição do projeto

Projeto consiste em um app de lista de contatos,
que conta com disponibilidade para os idiomas INGLES :us: e PORTUGUÊS :brazil:.


## 🙌 Recursos empregados 
* Jetpack Navigation
* Koin
* Coroutines
* Room
* Arquitetura MVVM
